## About

This is a personal engineering blog. I write about systems design, infrastructure, software architecture, and the craft of building reliable things.

The site is built with [Hugo](https://gohugo.io/), self-hosted, and intentionally minimal. No trackers, no analytics, no third-party dependencies beyond [Giscus](https://giscus.app/) for comments.

### Colophon

- **Generator:** Hugo (static site)
- **Hosting:** Self-hosted with Caddy
- **Comments:** GitHub Discussions via Giscus
- **Typography:** System serif for body, system sans for headings
- **Source:** Available on [GitHub](https://github.com/obarbier/tech.byobarbier.com)
